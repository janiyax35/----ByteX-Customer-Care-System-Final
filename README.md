## ğŸ”„ API Reference

The ByteX Customer Care System provides a comprehensive RESTful API for integration with other systems.

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/tickets` | GET | Retrieve all tickets |
| `/api/tickets/:id` | GET | Get specific ticket details |
| `/api/tickets` | POST | Create a new ticket |
| `/api/tickets/:id` | PUT | Update ticket information |
| `/api/customers` | GET | Retrieve customer information |
| `/api/agents` | GET | List all support agents |
